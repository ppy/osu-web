## TeamStatistics

```json
{
  "play_count": 123,
  "ranked_score": 63742331,
  "performance": 54634,
  "rank": 1,
  "ruleset_id": 0,
  "team_id": 1
}
```

A dictionary containing gameplay statistics of a team, for a certain ruleset.

| Field        | Type    | Description |
|--------------|---------|-------------|
| play_count   | integer |             |
| ranked_score | integer |             |
| performance  | integer |             |
| ruleset_id   | integer |             |
| team_id      | integer |             |
