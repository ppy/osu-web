## TeamMember
```json
{
  "created_at": "2026-01-22T019:05:15+00:00",
  "user": {
    "id": 2,
    "username": "peppy",
    "profile_colour": "#3366FF",
    "avatar_url": "https://a.ppy.sh/2?1519081077.png",
    "country_code": "AU",
    "is_active": true,
    "is_bot": false,
    "is_deleted": false,
    "is_online": true,
    "is_supporter": true
  }
}
```

Represents a member of a team.

| Field      | Type                    | Description                                  |
|------------|-------------------------|----------------------------------------------|
| created_at | [Timestamp](#timestamp) | Join date and time                           |
| user       | [User](#user)           | The actual user connected to this TeamMember |
